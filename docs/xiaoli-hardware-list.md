# 小李 ESP32 硬件采购建议

基于 78/xiaozhi-esp32 项目的硬件生态和搭建经验，目标是自己搭一套能稳定跑起来、价格尽量低、后续可扩展的小李 AI 语音设备。

## 推荐方案

优先选择散件 DIY：

```text
ESP32-S3-DevKitC-1 N16R8
+ INMP441 数字麦克风
+ MAX98357A I2S 数字功放
+ 4Ω/8Ω 2-3W 腔体喇叭
+ 面包板和杜邦线
```

这个组合性价比最高，资料最多，也最接近官方硬件搭建指南。第一次搭建不建议直接买贵的 M5Stack、ESP32-S3-BOX-3，也不建议优先用 ESP32-C3，因为 C3 扩展余量小，并且不支持离线语音唤醒。

## 必买清单

| 硬件 | 推荐规格 | 数量 | 参考价 |
| --- | --- | ---: | ---: |
| 主控开发板 | ESP32-S3-DevKitC-1 N16R8，16MB Flash + 8MB PSRAM，已焊排针 | 1 | 30-40 元 |
| 数字麦克风 | INMP441，已焊排针版，建议多买 1 个备用 | 2 | 6-9 元/个 |
| 数字功放 | MAX98357A I2S 功放，已焊排针 | 1 | 3-6 元 |
| 喇叭 | 4Ω/8Ω，2-3W 腔体喇叭 | 1 | 2-10 元 |
| 面包板 | 400 孔，或两块小面包板拼接 | 1 套 | 3-10 元 |
| 杜邦线/跳线 | 公对公、母对母、公对母都准备一些 | 1 套 | 5-10 元 |
| Type-C 数据线 | 必须是数据线，不是纯充电线 | 1 | 可用现有 |

不带屏幕时，整体预算大概 60-80 元。加上 OLED、按键和备用件后，大概 80-110 元。

## 可选硬件

| 硬件 | 推荐规格 | 是否建议一开始买 |
| --- | --- | --- |
| OLED 屏 | SSD1306 I2C，128x32 或 128x64 | 可以买，便宜且方便看状态 |
| 轻触按键 | 6x6mm，2 脚直插更省心 | 可以买，用于音量、唤醒、中断 |
| ML307R 4G 模块 | Cat.1，AT 固件版 | 先别买，除非明确要脱离 Wi-Fi 使用 |
| 锂电池/充电模块 | 取决于外壳和板子 | 第二阶段再考虑 |
| 外壳 | 3D 打印或现成外壳 | 跑通功能后再买 |

```text
MAX98357A I2S 数字功放 已焊排针

8欧3W 腔体喇叭
SSD1306 IIC OLED 128x32
400孔面包板 杜邦线套装
6x6 轻触开关 2脚
```

## 关键避坑

1. 主控一定优先买 N16R8，也就是 16MB Flash + 8MB PSRAM。
2. 不建议第一次买 N8R2。它也能跑，但需要额外改 PSRAM 模式、Flash 大小和分区表，还要重新编译固件。
3. INMP441 尽量买已焊排针版，并多买一个备用。VDD 和 GND 接反很容易烧。
4. MAX98357A 也买已焊排针版，减少焊接问题。
5. 喇叭买带腔体的，裸喇叭效果通常差很多。
6. Type-C 线必须支持数据传输，否则可能无法烧录。
7. 先用 Wi-Fi 跑通，不要一开始就加 4G 模块，4G 会增加供电、天线干扰和调试复杂度。

## 替代方案

| 方案 | 参考价 | 优点 | 缺点 | 建议 |
| --- | ---: | --- | --- | --- |
| 散件 DIY S3 | 60-110 元 | 最便宜，资料多，适合学习 | 接线多，需要调试 | 首选 |
| 虾哥 Xmini-C3 | 57-79 元 | 便宜，集成度高 | 不支持离线语音唤醒，扩展弱 | 只想省事可选 |
| 酷世 DIY ESP32-S3-SP-V3 | 119 元起 | S3，屏幕、电池接口更完整 | 比散件贵 | 想少接线可选 |
| 立创实战派 ESP32-S3 | 约 148 元 | 官方支持固件，屏幕可显示对话，可语音唤醒 | 成本更高 | 预算够且想省心可选 |
| ESP32-S3-BOX-3 / M5Stack CoreS3 | 250-300 元左右 | 做工和集成度好 | 性价比不高 | 不建议入门首选 |

## 推荐购买顺序

第一批先买：

```text
ESP32-S3-DevKitC-1 N16R8
INMP441 x2
MAX98357A
腔体喇叭
面包板
杜邦线
Type-C 数据线
```

跑通语音对话后，再买：

```text
SSD1306 OLED
轻触按键
外壳
电池相关模块
```

最后再考虑：

```text
ML307R 4G 模块
更大的 LCD/触摸屏
自定义 PCB
```
